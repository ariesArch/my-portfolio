# What is Scope?

variable/function တွေကို ကြေငြာပြီး ခေါ်ယူအသုံးပြု ခွင့်ကို သတ်မှတ်ထားတဲ့ ရပ်ဝန်းကို scope လို့ခေါ်ပါတယ်။
variable/function တွေရဲ့ visibility , accessibility တွေကို သတ်မှတ်ခြင်းလို့လည်း မှတ်ယူနိုင်ပါတယ်။
JavaScript မှာ scope 3 မျိုးရှိပါတယ်။

1. Global Scope 
2. Function/Local Scope
3. Block Scope
